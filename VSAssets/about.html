[[APP NAME]] aims to entertain and educate early-readers through a simple game where the hero collects items beginning with each letter of the alphabet.

Two game modes determine how the player is prompted (top left corner of the game screen), for instance:

  • [[LOCALIZED TEXT]] (Words): Find the Apple
  • [[LOCALIZED TEXT]] (Letters): What starts with A in [[LOCALIZED LANGUAGE ENGLISH NAME]]?

Tap the prompt in the upper left corner to see the picture of the current object needed, then tap the picture to hide the hint.

Scoring is as follows:

  • 25 points for each object retrieved
  • -1 point for each incorrect object touched
  • -5 points each time current object appears and is not retrieved
